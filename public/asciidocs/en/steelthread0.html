Steel Thread Zero

  1. Open the application launcher console.

  2. From the Create application screen, add a name for your REST endpoint application and click Continue.

  3. From the Target Environment screen, select "Code locally, Build and Deploy".

  4. From the Mission list, select CRUD.

  5. From the Runtime list, select Node.js and click the blue arrow to continue.

  6. Authorize your GitHub provider and click the blue arrow to continue.

  7. Review and click Set Up Application. You’ll see a confirmation screen when setup is complete.
